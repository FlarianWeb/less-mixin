# Spacing Mixins

Миксины для управления промежутками между элементами в flexbox и grid контейнерах.

## Содержание

- [gap](#gap)

---

## gap

**Назначение:** Определяет промежутки (отступы) между элементами в flexbox или grid контейнере. Управляет расстоянием между рядами и колонками, не затрагивая внешние края контейнера. Современная альтернатива margin для создания отступов в grid/flex layouts.

**Базовые миксины:**
- `.gap(@value)` - промежуток для обеих осей (px, rem, % и т.д.)
- `.gap-x(@value)` - промежуток между колонками (горизонтальный, column-gap)
- `.gap-y(@value)` - промежуток между рядами (вертикальный, row-gap)

**Глобальные пресеты gap:**
- `.gap-initial()` - начальное значение
- `.gap-inherit()` - наследование от родителя
- `.gap-unset()` - комбинация inherit/initial

**Глобальные пресеты gap-x:**
- `.gap-x-initial()` - начальное значение
- `.gap-x-inherit()` - наследование от родителя
- `.gap-x-unset()` - комбинация inherit/initial

**Глобальные пресеты gap-y:**
- `.gap-y-initial()` - начальное значение
- `.gap-y-inherit()` - наследование от родителя
- `.gap-y-unset()` - комбинация inherit/initial
