# Spacing Mixins

Миксины для управления отступами и промежутками: внешние отступы (margin), внутренние отступы (padding) и промежутки между элементами (gap).

## Содержание

- [gap](#gap)
- [margin](#margin)
- [padding](#padding)

---

## gap

**Назначение:** Определяет промежутки (отступы) между элементами в flexbox или grid контейнере. Управляет расстоянием между рядами и колонками, не затрагивая внешние края контейнера. Современная альтернатива margin для создания отступов в grid/flex layouts.

**Базовые миксины:**
- `.gap(@value)` - промежуток для обеих осей (px, rem, % и т.д.)
- `.gap-x(@value)` - промежуток между колонками (горизонтальный, column-gap)
- `.gap-y(@value)` - промежуток между рядами (вертикальный, row-gap)

**Глобальные пресеты gap:**
- `.gap-initial()` - начальное значение
- `.gap-inherit()` - наследование от родителя
- `.gap-unset()` - комбинация inherit/initial

**Глобальные пресеты gap-x:**
- `.gap-x-initial()` - начальное значение
- `.gap-x-inherit()` - наследование от родителя
- `.gap-x-unset()` - комбинация inherit/initial

**Глобальные пресеты gap-y:**
- `.gap-y-initial()` - начальное значение
- `.gap-y-inherit()` - наследование от родителя
- `.gap-y-unset()` - комбинация inherit/initial

## margin

**Назначение:** Устанавливает внешние отступы элемента - расстояние между границей элемента и соседними элементами. Используется для создания пространства вокруг элементов, управления расположением в потоке документа. Поддерживает как физические направления (top, right, bottom, left), так и логические (inline-start, inline-end) для поддержки RTL языков и вертикальных режимов письма.

**Базовые миксины:**
- `.m(@value)` - отступ со всех сторон
- `.mt(@value)` - margin-top (отступ сверху)
- `.mr(@value)` - margin-right (отступ справа)
- `.mb(@value)` - margin-bottom (отступ снизу)
- `.ml(@value)` - margin-left (отступ слева)
- `.mx(@value)` - margin по горизонтали (left + right)
- `.my(@value)` - margin по вертикали (top + bottom)
- `.ms(@value)` - margin-inline-start (отступ от начала строки, логическое свойство)
- `.me(@value)` - margin-inline-end (отступ от конца строки, логическое свойство)

**Глобальные пресеты для каждого миксина:**
- `.*-initial()` - начальное значение
- `.*-inherit()` - наследование от родителя
- `.*-unset()` - комбинация inherit/initial

## padding

**Назначение:** Устанавливает внутренние отступы элемента - расстояние между границей элемента и его содержимым. Используется для создания пространства внутри элементов, увеличения области фона и границ. Поддерживает как физические направления (top, right, bottom, left), так и логические (inline-start, inline-end) для поддержки RTL языков и вертикальных режимов письма.

**Базовые миксины:**
- `.p(@value)` - отступ со всех сторон
- `.pt(@value)` - padding-top (отступ сверху)
- `.pr(@value)` - padding-right (отступ справа)
- `.pb(@value)` - padding-bottom (отступ снизу)
- `.pl(@value)` - padding-left (отступ слева)
- `.px(@value)` - padding по горизонтали (left + right)
- `.py(@value)` - padding по вертикали (top + bottom)
- `.ps(@value)` - padding-inline-start (отступ от начала строки, логическое свойство)
- `.pe(@value)` - padding-inline-end (отступ от конца строки, логическое свойство)

**Глобальные пресеты для каждого миксина:**
- `.*-initial()` - начальное значение
- `.*-inherit()` - наследование от родителя
- `.*-unset()` - комбинация inherit/initial
