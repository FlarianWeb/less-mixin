## flex-basis

**Назначение:** Определяет базовый размер flex-элемента до того, как свободное пространство будет распределено в соответствии с flex-grow и flex-shrink. Это начальный размер элемента по главной оси, от которого начинается расчёт итогового размера.

**Базовый миксин:**
- `.flex-basis(@value)` - базовый размер (auto, px, %, rem и т.д.)

**Пресеты:**
- `.flex-basis-initial()` - начальное значение
- `.flex-basis-inherit()` - наследование от родителя
- `.flex-basis-unset()` - комбинация inherit/initial
- `.flex-basis-auto()` - автоматический размер (по умолчанию)
- `.flex-basis-full()` - 100% (полная ширина/высота)

## flex-direction

**Назначение:** Определяет направление главной оси flex-контейнера. Устанавливает направление, в котором размещаются flex-элементы внутри контейнера - по горизонтали или вертикали, в прямом или обратном порядке.

**Базовый миксин:**
- `.flex-direction(@value)` - направление (row, row-reverse, column, column-reverse)

**Пресеты:**
- `.flex-direction-initial()` - начальное значение
- `.flex-direction-inherit()` - наследование от родителя
- `.flex-direction-unset()` - комбинация inherit/initial
- `.flex-row()` - горизонтальное направление слева направо (по умолчанию)
- `.flex-row-reverse()` - горизонтальное направление справа налево
- `.flex-col()` - вертикальное направление сверху вниз
- `.flex-col-reverse()` - вертикальное направление снизу вверх

## flex-wrap

**Назначение:** Определяет, будут ли flex-элементы переноситься на новую строку/колонку, если не помещаются в одну линию. Управляет многострочным поведением flex-контейнера и направлением переноса.

**Базовый миксин:**
- `.flex-wrap(@value)` - режим переноса (nowrap, wrap, wrap-reverse)

**Пресеты:**
- `.flex-wrap-initial()` - начальное значение
- `.flex-wrap-inherit()` - наследование от родителя
- `.flex-wrap-unset()` - комбинация inherit/initial
- `.flex-nowrap()` - все элементы в одну линию, без переноса (по умолчанию)
- `.flex-wrap()` - перенос на новые строки по мере необходимости
- `.flex-wrap-reverse()` - перенос в обратном направлении

## flex-grow

**Назначение:** Определяет способность flex-элемента расти и занимать свободное пространство в flex-контейнере. Значение задаёт пропорцию, в которой элемент будет увеличиваться относительно других flex-элементов при наличии свободного места.

**Базовый миксин:**
- `.flex-grow(@value)` - коэффициент роста (число, обычно 0-1 или больше)

**Пресеты:**
- `.flex-grow-initial()` - начальное значение
- `.flex-grow-inherit()` - наследование от родителя
- `.flex-grow-unset()` - комбинация inherit/initial
- `.flex-grow()` - элемент может расти (flex-grow: 1)
- `.flex-grow-0()` - элемент не может расти (по умолчанию)

---

## flex-shrink

**Назначение:** Определяет способность flex-элемента сжиматься при нехватке места в flex-контейнере. Значение задаёт пропорцию, в которой элемент будет уменьшаться относительно других flex-элементов при недостатке пространства.

**Базовый миксин:**
- `.flex-shrink(@value)` - коэффициент сжатия (число, обычно 0-1 или больше)

**Пресеты:**
- `.flex-shrink-initial()` - начальное значение
- `.flex-shrink-inherit()` - наследование от родителя
- `.flex-shrink-unset()` - комбинация inherit/initial
- `.flex-shrink()` - элемент может сжиматься (flex-shrink: 1, по умолчанию)
- `.flex-shrink-0()` - элемент не может сжиматься

---

## order

**Назначение:** Определяет порядок отображения flex-элемента или grid-элемента внутри контейнера. Позволяет изменить визуальный порядок элементов без изменения DOM-структуры. Элементы сортируются по возрастанию значения order, затем по порядку в исходном коде.

**Базовый миксин:**
- `.order(@value)` - порядковый номер (целое число, может быть отрицательным)

**Пресеты:**
- `.order-initial()` - начальное значение
- `.order-inherit()` - наследование от родителя
- `.order-unset()` - комбинация inherit/initial
- `.order-first()` - в начало списка (order: -9999)
- `.order-last()` - в конец списка (order: 9999)
- `.order-none()` - обычный порядок (order: 0, по умолчанию)
