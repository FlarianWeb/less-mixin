## width

**Назначение:** Устанавливает ширину элемента. Определяет горизонтальный размер блока контента. Поддерживает фиксированные размеры (px, rem), относительные (%, viewport units) и семантические значения (auto, min-content, max-content, fit-content). Включает поддержку новых динамических viewport единиц (dvw, lvw, svw) для корректной работы на мобильных устройствах.

**Базовый миксин:**
- `.w(@value)` - ширина (px, %, vw, auto и т.д.)

**Пресеты:**
- `.w-initial()` - начальное значение
- `.w-inherit()` - наследование от родителя
- `.w-unset()` - комбинация inherit/initial
- `.w-auto()` - автоматическая ширина (по умолчанию)
- `.w-full()` - 100% ширины родителя
- `.w-screen()` - 100vw (ширина viewport)
- `.w-dvw()` - 100dvw (динамическая ширина viewport)
- `.w-dvh()` - 100dvh (динамическая высота viewport)
- `.w-lvw()` - 100lvw (большая ширина viewport)
- `.w-lvh()` - 100lvh (большая высота viewport)
- `.w-svw()` - 100svw (малая ширина viewport)
- `.w-svh()` - 100svh (малая высота viewport)
- `.w-min()` - min-content (минимальная ширина контента)
- `.w-max()` - max-content (максимальная ширина контента)
- `.w-fit()` - fit-content (подогнать под контент)

## size

**Назначение:** Одновременно устанавливает ширину и высоту элемента. Удобный shorthand для задания размеров квадратных элементов или элементов с разными размерами по осям. Поддерживает перегрузку миксина - можно передать одно значение для обеих осей или два отдельных значения для ширины и высоты.

**Базовые миксины:**
- `.size(@value)` - одинаковая ширина и высота
- `.size(@width; @height)` - разные ширина и высота

**Пресеты:**
- `.size-initial()` - начальное значение для обеих осей
- `.size-inherit()` - наследование от родителя для обеих осей
- `.size-unset()` - комбинация inherit/initial для обеих осей
- `.size-auto()` - автоматические размеры (auto x auto)
- `.size-full()` - 100% x 100% (заполнить родителя)
- `.size-screen()` - 100vw x 100vh (заполнить viewport)
- `.size-screen-d()` - 100dvw x 100dvh (динамический viewport)
- `.size-screen-l()` - 100lvw x 100lvh (большой viewport)
- `.size-screen-s()` - 100svw x 100svh (малый viewport)
- `.size-min()` - min-content x min-content
- `.size-max()` - max-content x max-content
- `.size-fit()` - fit-content x fit-content
